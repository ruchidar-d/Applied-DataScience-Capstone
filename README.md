# IBM Data Science Professional Project
🚀 SpaceX Falcon 9 First Stage Landing Prediction

This project is a complete end-to-end Data Science analysis of SpaceX Falcon 9 launch data, including data collection, web scraping, data wrangling, SQL analysis, visualization, and exploratory data analysis.

The objective of this project is to analyze SpaceX launch data and gain insights into launch success rates and first-stage landing outcomes.



🎯 Project Objectives

Collect real SpaceX launch data via API and web scraping
Perform data wrangling and cleaning
Conduct Exploratory Data Analysis (EDA) using SQL and visualizations
Build interactive dashboards and maps
Train and evaluate machine learning models for binary classification (landed vs. not landed)




🛠️ Technologies & Tools Used

Python 3
Pandas, NumPy
Matplotlib, Seaborn, Plotly
Folium (maps)
Plotly Dash (interactive dashboard)
Scikit-learn (ML models & GridSearchCV)
BeautifulSoup + Requests (web scraping)
SQLite (SQL analysis)
Jupyter Notebooks





🚀 Key Findings

Success rate improved significantly after 2017 with booster reuse and landing technology
Launch sites show different success patterns (CCAFS SLC-40 vs KSC LC-39A vs VAFB SLC-4E)
Payload mass and orbit type strongly influence landing success
Falcon 9 Block 5 boosters show the highest reliability
Best performing model: Logistic Regression / SVM (~83–84% accuracy on test set)





📊 Interactive Visualizations Included

Folium maps with launch site locations and proximity to coastline/highways
Dash app with:
Success/failure pie charts by launch site
Payload mass vs success scatter plot with booster version coloring

SQL queries showing success counts, booster performance, and payload statistics
